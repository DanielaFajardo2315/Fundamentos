Fundamentos de desarrollo web: HTML y CSS

¿Que es el HTML?

Lenguaje de marcado que estructura el contenido de una página web. Basado en etiquetas que pueden ser semánticas o no semánticas.

¡No es un lenguaje de programación!

Semántica en HTML

Etiquetas básicas del HTML

Etiqueta Descripción
h1, h2, h3, h4, h5, h6 Títulos y subtítulos en HTML
p Párrafos
img Para mostrar las imágenes (src, alt)
ul, ol Etiquetas para definir listas desordenadas y ordenas
div Divisores de contenidos- dar estilos
a Redireccionar a un link fuera o dentro de nuestra página

Formularios en HTML

Son una herramienta fundamental para recopilar datos de un usuario, a través de campos como:

  • Textos
  • Casillas de texto - áreas de texto
  • Listas desplegables

Ejemplo: Formulario de inicio de sesión



Enlaces en HTML

Permiten navegar a otras páginas web (documentos de HTML), secciones del mismo sitio o de la misma página. Descargar un archivo al hacer click en el enlace.
